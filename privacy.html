Privacy Information - LucOS Authentication

Please note that this information is only applicable to the LucOS authentication service. Individual LucOS apps may go beyond what's covered here.

Data Storage

The following data is logged and held temporarily on LucOS servers for the purposes of IT Security:

  • IP Addresses
  • Browser information (known as a "User Agent String")
  • A list of pages visited

Cookie Usage

LucOS Authentication sets cookies for the following purposes:

  • Remembering which LucOS app to return you to following login
  • Remembering who you are so that you don't need to login multiple times across LucOS apps
If you disable cookies in your browser, hopefully login should still work. However, you'll need to login again for every LucOS app. Note that no guarantees are made, as this flow is rarely tested.

Retrival of data found elsewhere

LucOS Authentication attempts to match up information regarding the account you've used to login, with contacts stored elsewhere in the LucOS system. This process doesn't create new contacts or store additional data on existing contacts. Some LucOS apps ensure such a match has been found before considering a login successful.

Veracity of this document

Whilst attempts were made to ensure information contained in this document was correct at the time it was authored, no guarantees are made as to its veracity. Also, changes to the system may be made without this document being updated, leading to potential inaccuracies.
